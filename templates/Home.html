Text to Image 

This is a simple api with which you can automate your social media posts, blogs, etc.
The base url is https://texttoimg.herokuapp.com

Main URL

https://texttoimg.herokuapp.com

Get the text with simple white background color ?
(main url)/img/?text=text

Exmaple :-

(main url)/img/?text=this is an exmaple
Get the text with custom size & background color ?
(main url)/custom-img/?text=text&fontsize=value&fontcolor=​value&width=​value&height=​value&bg=​value

Exmaple :-

(main url)​/custom-img/?text=any text&fontsize=20&fontcolor=black&width=200&height=70&bg=white
Get the text with a random backgroung image ?
(main url)​/photo/?text=text

Exmaple :-

(main url)/photo/?text=any text
Get the text with a ​specific backgroung image ?
(main url)​​/custom-photo/?text=text&bg=value

Exmaple :-

(main url)​/custom-photo/?text=any text&bg=india
